## 2024-06-22 - [Dynamic ARIA labels for icon-only toggle buttons]
**Learning:** Sighted users rely on icon semantics and tooltips for understanding stateful icon-only buttons (like `PanelRightClose`/`PanelRightOpen`), but screen reader users completely miss the context if `aria-label` is missing or static. Furthermore, dynamic toggles need changing `aria-label`s to indicate current state correctly (e.g. "Close properties sidebar" instead of just "Toggle properties"). Keyboard focus styling (e.g. `focus-visible:ring-2`) is also essential for visual indication during tab navigation.
**Action:** When adding icon-only toggle buttons, always define dynamic `aria-label` and `title` attributes that update according to the active/inactive state of the toggle, and ensure that focus outlines are explicit using `focus-visible`.
